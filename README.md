# Modular-BiAMP
Sample Modular Bi-AMP designed to work with Technics HD501 speakers (bc ampilifre in mine died). Project in active development.

## Modules
- Crosover PCB - Simple Active opamp based crosover providing 80db/dec attenuation. Components selected for 20 Hz to 3kHz passband for low freq part and 3kHz to 20kHz passband for high freq part.
- Power Amp PCB - Amplifire (AB-class) board using LM3886 for low frequence part (woofers) and LM4766 for high Frequency parts (tweeters)
